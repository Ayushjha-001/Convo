# Convo 💬

A full-stack real-time chat application built with the MERN stack. Convo enables users to communicate instantly through a modern interface with secure authentication, real-time messaging, and online user presence tracking.

## ✨ Features

* 💬 Real-time messaging with Socket.io
* 🟢 Online user status indicators
* 🔐 Authentication & Authorization using JWT
* 🍪 Secure HTTP-only cookie authentication
* ⚡ Global state management with Zustand
* 🎨 Responsive UI built with Tailwind CSS and DaisyUI
* 🛡️ Protected routes and secure backend APIs
* ❌ Client-side and server-side error handling
* ☁️ Full-stack deployment on Render

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* DaisyUI
* Zustand
* React Router DOM
* Socket.io Client

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Socket.io
* JWT Authentication
* bcryptjs
